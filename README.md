## 프로젝트 명
마트루 하이브리드앱

## 프로젝트 소개
웹뷰를 통한 하이브리드앱
<br><br>

## 프로젝트 기간
2021.01.25 ~ 2021 03.02<br><br>

## 기능
* FCM을통한 푸쉬서비스 
* Firebase DynamicLink를 통한 앱 이동 
* Glide를 통한 동적 스플래쉬 
* 하이브리드 앱 성능개선(캐시 사이즈 UP)
* okHttp를 통한 버전체크 및 앱 업데이트 알림 
* Corutine을 통한 비동기 제어
<br><br>



## 기술스택
* Android (Kotlin)
* Firebase Cloud Messaging(FCM)
* Firebase Dynamic Link
* Glide(동적 스플래쉬)
* OKHttp
* Coroutine 
* View Binding

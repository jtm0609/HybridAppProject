## 프로젝트 명
마트루 하이브리드앱

## 프로젝트 소개
웹뷰를 통한 하이브리드앱
<br><br>

## 프로젝트 기간
2021.01.25 ~ 2021 03.23<br><br>

## 기능
* FCM을통한 푸쉬서비스 
* Firebase DynamicLink를 통한 앱 이동
* Http Cache 적용 및 앱 캐시사이즈 업그레이드
* InApp 업데이트
* PlayStore 업데이트

<br><br>



## 기술스택
* Android (Kotlin)
* Firebase Cloud Messaging(FCM)
* Firebase Dynamic Link
* Retrofit
* Coroutine 
* View Binding
